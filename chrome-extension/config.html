SubPirate Extension Configuration

Extension Settings

Configure your SubPirate Chrome extension with the proper environment variables.

Installation Instructions

  1. Fill in the configuration form above with your Supabase and app details
  2. Click "Save Configuration" to store these settings
  3. If prompted, download and replace the background.js file
  4. Load the extension in Chrome:
    • Open Chrome and go to chrome://extensions/
    • Enable "Developer mode" (toggle in the top right)
    • Click "Load unpacked" and select the extension directory
  5. The extension should now appear in your Chrome toolbar

Server Integration

To enable the extension to communicate with your SubPirate app, you'll need to add API endpoints to your server.

Check the server/integration.js file for instructions on how to integrate these endpoints with your existing server.

// In your server.js or main server file:
import { setupExtensionRoutes } from './chrome-extension/server/integration';

// Initialize your Express app
const app = express();

// Set up extension routes
setupExtensionRoutes(app);
